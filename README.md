# Abstraction Site

Quick start

1. Install dependencies

```bash
npm install
```

2. Create a `.env` file with:

```
VITE_SUPABASE_URL=https://your-project.supabase.co
VITE_SUPABASE_KEY=public-anon-key
```

3. Run the dev server

```bash
npm run dev
```

4. Testing

Run the test suite with Vitest:

```bash
npm test
```
